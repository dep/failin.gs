failin.gs by dep

failin.gs seeks to answer that age old question that we've all asked ourselves at one point. "What is wrong with me?"

Install

First, get Ruby 1.9.1 up and running. % [sudo] gem install rvm % rvm-install % rvm install 1.9.1 % rvm 1.9.1 Then, prepare the application. % git submodule update --init % script/bundle % cp config/database.yml.example config/database.yml % bin/rake db:create:all db:migrate
